SOBRE O PROJETO

Possui a missão de encurtar a jornada do estado atual para o estado desejado. Para isso, temos um time de profissionais consultores de carreiras que farão uma entrega diferenciada, buscando encaixar as virtudes e competências dos jovens com as vagas disponíveis das empresas parceiras. O Programa não promete vaga de emprego, garante encurtar a jornada entre quem precisa contratar e os candidatos que possuem competência para as para as vagas.

Como Funcuiona o projeto ?

O Jovem Trabalhador é um programa sem fins lucrativos que tem como objetivo principal os jovens que buscam no mercado de trabalho e se desenvolvem profissionalmente. Nossa missão é unir os candidatos às empresas que recrutamento de talentos para seu quadro de colaboradores.

Contratação do menor aprendiz ?

Após o cadastro, o Jovem Trabalhador entrará em contato, em uma entrevista avaliaremos o perfil do candidato. Com a aprovação do perfil e das competências o jovem será encaminhado para processo seletivo.

Custo do Processo seletivo ?

Não há custo para participar do processo seletivo. Este cadastro é gratuito, não há qualquer tipo de cobrança para o jovem se cadastrar ou ser encaminhado.
